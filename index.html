    System Architecture & Docs ▼

    1. Raw Audio Streaming: Client streams raw binary audio blobs directly to avoid Base64 RAM spikes and mobile OOM crashes.

    2. local-first Execution: Whisper Small Q8 (STT) and MedGemma 4B IT (LLM) models run entirely offline on consumer hardware via @qvac/sdk.

    3. Decentralized Mesh: Edge nodes discover and announce under med-scribe-v1 using a Hyperswarm DHT P2P network bridge.

    4. Privacy Architecture: Clinician recordings and notes never transit to external cloud servers, satisfying HIPAA PHI criteria.
